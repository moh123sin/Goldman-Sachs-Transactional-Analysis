This project performs transactional data analysis for Goldman Sachs using Python and pandas. I loaded the raw transaction file, cleaned and transformed the data, and then explored key metrics such as transaction volume, values, client segments and product types to understand overall activity patterns.
The notebook walks through the full workflow from data import and preprocessing to visualizations and summary tables. The final report highlights important trends, anomalies and high‑value segments, providing insights that could support risk monitoring, compliance checks and business decision‑making for the bank.
